No Bullshit Git

Why Take This Course?

No Bullshit Git shows you a better way of using Git. It's here to help you harness the full power of Git in real-world development and teach you the techniques and commands that will make your work easier.

  • Significantly speed up your development
  • Fix and undo mistakes easily
  • Navigate and manipulate your Git history effortlessly
  • Become your team’s go-to Git person
  • Know how to fix things when they break
  • Never delete and then clone your repo to fix a problem again
  • Untangle thorny Git merges
  • Go beyond copy-and-paste Stack Overflow solutions
  • Be confident in your Git usage
  • Stop needing to ask other developers for help

What You're Getting

So what does a practical guide to learning Git actually look like?

No cruft

No Bullshit Git has been slimmed down to remove the fluff. What's left are the important concepts without any filler.

Quick and to the Point

You've got better things to do than listen to a non-stop lecture. Lessons are five minutes or less.

No Ivory Towers

The last thing any developer needs is another lesson on the "theory of Git". No Bullshit Git focuses on things you'll use day-to-day.

Context

Learn when and why to use commands, not just how to use them.

Exercises Included

No Bullshit Git includes real-world exercises you can download and try out on your own.

Curation

It’s hard to know what you don't know. No Bullshit Git gives you new ways to think about and use Git.

Go Deep

Learn the core concepts of Git that help you really "get" it rather than relying on trial and error.

Beyond the Basics

Most Git tutorials are written for complete beginners, and a few are written for rocket surgeons. No Bullshit Git is for people in the middle.

Developer-Friendly

No Bullshit Git is created from the experience of a developer teaching other professional developers how to use Git effectively.
